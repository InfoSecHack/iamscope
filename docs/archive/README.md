# Archived Benchmark and Checkpoint Material

This folder contains historical benchmark planning, checkpoint, protocol,
readiness, changelog, and packaging-audit material moved out of the first-read
reviewer path for public research-preview readability.

It is not the current public reviewer entrypoint. Public reviewers should start
with `README.md`, `docs/START_HERE.md`, and
`docs/specs/supported-unsupported-evidence-matrix.md`.

Archived docs may contain historical context, superseded instructions, or older
packaging/release planning notes. Current evidence and reviewer guidance live in
the first-read documents above.

`BENCHMARK_STATUS.md` remains the root benchmark entrypoint. This archive preserves evidence-program history without adding production-readiness, broad-correctness, exploitability, pass/fail, or composite-score claims.
