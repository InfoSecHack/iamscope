# IAMScope Rebuild — Deviations Ledger

This file tracks every place where the executing agent (the agent) deviated from
the literal rebuild plan. Each deviation has an ID, the session that raised it,
a one-sentence summary, and a disposition. Dispositioned entries do not need
further review unless the maintainer explicitly wants to revisit the call.

This file lives in the repo root so the audit trail travels with every tarball
from S04 onward.

## Legend

- **Closed** — decision made, action taken, no further work.
- **Deferred post-V1** — acknowledged but not fixed; slot into a later release.
- **Open** — needs the maintainer input or a future session to resolve.
